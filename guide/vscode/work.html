1,vue 进入/离开 当前页面触发的函数代码

进入：

mounted(){};

离开：

destroyed(){};

说明：destroyed是与methods、mounted同层级的

2,vue 利用Vue中keep-alive，快速实现页面缓存,就是不完全关闭该页面

初次进入时：created > mounted > activated；退出后触发 deactivated

// 进入
mounted () {
  this.loading = true;
},
// 进入页面的时候请求继续调接口
activated () {
  const that = this;
  if (that.symbol && that.market) {
    that.interval = setInterval(function () {
      that.leftBackcolor = "leftBack";
      that.rightBackcolor = "rightBack";
      that.getData(that.symbol, that.market);
    }, 3000);
  }
},
// 离开页面的时候终止调用接口
deactivated () {
  const that = this;
  clearInterval(that.interval);
  that.interval = null;
},
// 如果只有单独这个order 页面，且点叉关闭的时候，走这个销毁函数
destroyed () {
  const that = this;
  clearInterval(that.interval);
  that.interval = null;
},
